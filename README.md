# advanced-note-app

This is an basic evernote clone which is used to take notes having features like
1. Register
2. Login
3. Add new note
4. Update note
5. Delete note
6. List all the notes of current user
7. Search list of current user notes

# Stack used
1. Node
2. Express
3. Graphql
4. TypeORM
5. JWT
6. PostgreSQL
7. Reactjs
8. Apollo client
